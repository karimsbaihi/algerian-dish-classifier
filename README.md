# algerian-dish-classifier
Algerian Dish Classifier that classifies 15 disctinct algerian dishes 
## the dishes are:
Couscous
Chakhchoukha
Rechta
Mhadjeb
Bourek
Chorba frik
Harira
Dolma
Tajine zitoune
Karantika (Garantita)
Mhalbi
Makroud
Baghrir
Zlabia
Kalb el louz
These are visually distinct (soup vs pastry vs stuffed vegetable vs sweet vs flatbread)
